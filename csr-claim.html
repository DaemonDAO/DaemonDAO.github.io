CSR Claims interface

Canto implements a fee split model called Contract Secured Revenue, or CSR. CSR allows you to claim a percentage of all transaction fees (initially 20%) paid by users when interacting with your smart contracts.

See Canto docs for instructions on how to register a smart contract for CSR on deployment, or check the DigiDaemons contract code for a working example.
